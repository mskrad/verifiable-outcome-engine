**[SLIDE 2 — Problem]**

Today, the explorer shows you the transaction.
But it doesn't show you the decision.
You can see that a winner was written on-chain.
But you don't know if the rules were locked before the draw, who was actually in the participant list, or whether the backend was changed before committing the result.
So the transaction is public.
But the decision is still a black box.

---

**[SLIDE 3 — Value Prop]**

VRE changes it.
Before the draw happens, we commit the outcome rules on-chain.
The participant list, the selection formula, and the constraints are locked before resolution.
After that, anyone can replay the result independently.
Users can check it.
Competitors can check it.
Auditors can check it.
The backend is no longer the source of truth.

Then VRE resolves the outcome and returns a Solana transaction signature.
Anyone can take that signature and replay the result.

Winner correct.
