Use cases

Same engine. Four surfaces.

Every card above uses the same deterministic kernel. The use-case label is descriptive, not architectural — VRE doesn't care what your draw is for.

🎟

Raffle

Weighted selection from a participant list. One winner (or N), seeded from a slot + nonce.

🪂

Airdrop

Multi-winner eligible-list selection with per-address weights. Useful for fair distributions to a long tail.

🔮

Prediction

Possible outcomes and their probabilities committed before the event resolves. Oracle-independent.

🎁

Loot

Weighted outcome table (legendary / rare / common). Identical kernel to raffle, different label.
