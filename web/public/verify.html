Independent verifier

Verify Outcome

Paste a transaction signature from a vre resolve call. The verifier fetches it from a public RPC, reconstructs the committed W3O1 rules, and replays the selection. For v3 artifacts it also surfaces formula, target, score, and stable order.

Anyone can run this. The public API replays the outcome from RPC data and returns a specific reason code.
