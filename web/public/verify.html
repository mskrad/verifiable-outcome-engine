Independent verifier

Verify Outcome

Paste a transaction signature from a vre resolve call. The verifier fetches it from a public RPC, reconstructs the committed rules, and replays the selection. If the on-chain result matches the replay, you get MATCH / OK.

Anyone can run this. The public API replays the outcome from RPC data and returns a specific reason code.
