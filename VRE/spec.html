01

Invariants

Five rules every VRE outcome obeys. Break any of them and the verifier returns a non-OK reason code.

  • Pre-commit. The full ruleset hash is committed on-chain before the resolve transaction is submitted.
  • Determinism. Given identical inputs, the kernel produces bit-identical outputs across Node, browser, and Deno.
  • Public sourcing. All inputs to the selection function are public Solana state — no off-chain oracles, no sealed inputs.
  • Independent replay. Any third party can reproduce the outcome from public RPC data alone.
  • Structural authorship. The operator cannot bias the result after commit, and provably cannot.

03

Determinism

The kernel is a small pure function: (ruleset_hash, seed) → outcome. It never reads system time, never makes a network call, never branches on environment.

kernel.ts · vre-core@1.0.0
// Pure. Stateless. Reproducible. export function select(ruleset: Ruleset, seed: Bytes32): Outcome { const h = sha256(canonicalize(ruleset)); // 32 bytes const rng = chacha20(concat(h, seed)); // stream const idx = weightedPick(ruleset.weights, rng); return { artifact_hash: toHex(h), winner: ruleset.participants[idx], selection: idx, }; }
04

Replay process

What the verifier does when you paste a signature:

  1. Fetch. Call getTransaction(sig) on a public RPC. Decode the resolve instruction.
  2. Resolve runtime. Look up the prior commit at the referenced runtime_id; pull the artifact hash.
  3. Reconstruct ruleset. Read the ruleset bytes from the program account, canonicalize, hash. Must match.
  4. Replay kernel. Run select() locally with the on-chain seed.
  5. Compare. If the locally-replayed winner equals the on-chain winner, return MATCH / OK.

07

Error codes

Every verification returns one of these reason codes. Codes are stable across SDK versions.

MATCH
Replay reproduces the on-chain outcome bit-for-bit.
PENDING
Commit found, draw not yet resolved. Try again after the resolution slot.
RULESET_MISMATCH
Artifact hash on chain differs from what the verifier reconstructed.
NO_COMMIT
No prior commit found for this resolve. Outcome is not verifiable.
REPLAY_DIVERGED
Kernel produced a different winner than the on-chain record.
RPC_UNREACHABLE
Could not fetch the signature. Try a different RPC endpoint.
SIG_NOT_FOUND
RPC returned no transaction for the supplied signature.
PROGRAM_MISMATCH
Transaction did not invoke the canonical VRE program.
08

Metaplex Agent Registry

VRE composes with the Metaplex Agent Registry as an external identity layer. Agent metadata and registry membership can be published independently, while VRE still commits W3O1 bytes on-chain and replays outcomes from public RPC data.

09

Sybil resistance

Optionally gate participants by World ID proof-of-personhood. The World proof is verified before the live raffle call is accepted, and the participant list that reaches W3O1 replay remains deterministic and public.
